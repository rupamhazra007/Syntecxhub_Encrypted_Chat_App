# 🔐 Encrypted Chat Application

## Project
Syntecxhub_Encrypted_Chat_App

## Overview
This project is a secure client-server chat application built using Python. Messages are encrypted using **AES encryption** before being sent over **TCP sockets**, ensuring secure communication between clients.

## Features
- AES Encryption
- TCP Socket Communication
- Multiple Client Support
- Pre-shared Key
- Message Logging

## Technologies
- Python
- Socket Programming
- Cryptography Library

## Run Project

Install dependency:

pip install cryptography

Start server:

python server.py

Run client:

python client.py
